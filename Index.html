Foundation

SEO Fundamentals

Search engines kis tarah kaam karte hain, ranking factors kya hain, aur SEO ki har major category (on-page, off-page, technical, local, international, e-commerce, AI/GEO) ka bunyadi taaruf.

SEO Kya Hai?

Website ko search engines (Google, Bing) me organic (bina paid ads) top rank dilwane ka process.

Search Engine Kaise Kaam Karta Hai

Crawling → Indexing → Ranking. Bots pages crawl karte, content ko index me store karte, phir query ke hisaab se rank karte hain.

Google Ranking Factors

Content quality, backlinks, UX/Core Web Vitals, on-page relevance, technical health aur E-E-A-T signals.

White Hat vs Black Hat

White Hat = guidelines follow karke long-term growth. Black Hat = shortcuts (keyword stuffing, cloaking) jo penalty laate hain.

On-Page SEO

Title, meta, headings, content, internal links — jo page ke andar control hota hai.

Off-Page SEO

Backlinks, mentions, social signals — website ke bahar authority build karna.

Technical SEO

Crawlability, indexing, speed, schema, site architecture ki health.

Local SEO

Google Business Profile aur "near me" searches ke liye local visibility.

International SEO

Hreflang, multi-language/multi-country targeting ke liye setup.

E-commerce SEO

Product/category pages, filters, structured data aur duplicate content handling.

AI SEO & GEO

ChatGPT, Gemini, Perplexity jaisi AI answer engines me visible hone ki optimization (Generative Engine Optimization).

Research

Keyword Research

User jo words search bar me type karta hai — unhe dhoondna aur unke peeche ki "intent" (irada) samajhna.

▸ Search Intent Types

Informational

"kya", "kaise", "why" — sirf jaankari chahiye. Example: "seo kya hai".

Commercial

Compare/review karne ki niyat. Example: "best seo tools 2026".

Transactional

Buy/signup karne ka irada. Example: "ahrefs pricing buy".

Navigational

Specific brand/site dhoondna. Example: "google search console login".

▸ Keyword Types

  • Short Tail — 1-2 words, high volume, high competition (e.g. "SEO")
  • Long Tail — 4+ words, specific, easier to rank (e.g. "SEO kaise sikhein beginners ke liye")
  • LSI Keywords — semantically related terms jo topic context deep karte hain
  • Topical Keywords — ek broad topic cluster ke sub-keywords
  • Question Keywords — "kaise", "kya", "kyun" se shuru hone wale queries
  • Seasonal Keywords — time/event specific spikes (e.g. "eid offers")

Core Web Vitals

LCP, INP, CLS — user experience/speed ke Google metrics.

Complete SEO Checklists

Har platform/scenario ke liye ready-to-use checklists.

New Website SEO Checklist SETUP

  • GSC + GA4 setup
  • XML sitemap submit
  • Robots.txt configure
  • SSL/HTTPS enable
  • Mobile responsiveness check
  • Site structure/navigation plan
  • Core Web Vitals baseline
  • 404 & redirect setup

Blog SEO Checklist CONTENT

  • Target keyword in title/H1
  • Meta title & description
  • Optimized URL slug
  • Internal + external links
  • Image alt text
  • FAQ schema (if applicable)
  • Readable formatting (H2/H3)
  • CTA at the end

Local SEO Checklist LOCAL

  • Google Business Profile complete
  • NAP consistency everywhere
  • Local citations built
  • Customer reviews strategy
  • Local keywords in content
  • City/area landing pages

WooCommerce SEO Checklist ECOM

  • Product schema markup
  • Unique product descriptions
  • Category page optimization
  • Image compression
  • Duplicate content (filters) handled
  • Breadcrumbs enabled

WordPress SEO Checklist CMS

  • SEO plugin configured (Rank Math/Yoast)
  • Permalink structure clean
  • Caching plugin active
  • Image optimization
  • Broken plugin/theme cleanup
  • Schema markup enabled

Shopify SEO Checklist ECOM

  • Theme speed optimization
  • Alt text on product images
  • URL handles cleaned
  • Collection page SEO
  • App bloat minimized
  • Structured data verified

Technical SEO Checklist TECH

  • Crawl errors resolved
  • Canonical tags correct
  • Core Web Vitals passing
  • Sitemap up to date
  • Structured data valid
  • No orphan pages

Content SEO Checklist CONTENT

  • Search intent matched
  • E-E-A-T signals present
  • Topical depth covered
  • Internal linking done
  • Content freshness maintained
  • Multimedia included
